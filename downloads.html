Téléchargements

Premiers pas

Vous êtes un nouvel utilisateur ? Consultez nos pages Premiers pas, FAQ et Tutoriels sur le wiki.
Sinon, vous pouvez jeter un œil au journal des modifications.

Vous pourriez également vouloir explorer quelques jeux. Les jeux définissent le gameplay de base du moteur, en utilisant des scripts Lua. Chaque jeu propose des objectifs différents, comme la survie, la construction ou le PvP. Vous pouvez ensuite ajouter des modifications (mods) pour personnaliser davantage votre expérience.

Windows

Compatible avec Windows 8, 8.1 et 10. Peut fonctionner sur des versions plus anciennes, mais non officiellement supporté.

  • Luanti 5.11.0 - version portable, 64 bits (recommandé)
  • Luanti 5.11.0 - version portable, 32 bits

Besoin d'aide ? Consultez les instructions pour Windows.

Vous pouvez également obtenir la version de développement la plus récente via les builds communautaires. Ces versions incluent de nouvelles fonctionnalités (mais peuvent être moins stables).

Android

Android 6 ou une version ultérieure est recommandé.

Note : Nous déconseillons d'utiliser les versions non officielles disponibles sur le Play Store. Elles peuvent contenir des publicités excessives ou des logiciels espions, ou être distribuées sous des termes propriétaires.

Linux

Paquets

Ces paquets peuvent parfois être obsolètes. Si c'est le cas, envisagez de compiler le logiciel à partir des sources.

  • Flatpak (toutes distributions) - Stable
  • Ubuntu - PPA Stable ou PPA Journalier
  • Debian - Flatpak ou compiler depuis les sources
  • Arch Linux - Stable
  • Fedora - Stable

FreeBSD

Gestionnaire de paquets

Sur FreeBSD 9.1 et versions plus récentes, vous pouvez utiliser les paquets officiels précompilés : pkg install minetest

Compilation via les ports

Vous pouvez également compiler Luanti et choisir facilement les options de compilation grâce aux dialogues des ports FreeBSD.

    cd /usr/ports/games/minetest
    make install

macOS

macOS 11.3 ou une version plus récente est recommandé. Peut fonctionner sur des versions plus anciennes si compilé à partir des sources.

  • Signé pour Apple Silicon - Luanti 5.11.0 - Application
  • Signé pour Intel 64 bits - Luanti 5.11.0 - Application
  • Utilisez Homebrew pour obtenir la version stable :
    brew install minetest
  • MacPorts propose également un port de la version stable :
    sudo port install minetest

Code source

Obtenez la dernière version stable ou en développement sur GitHub.

Consultez le README pour plus de détails sur la compilation de Luanti à partir des sources.
