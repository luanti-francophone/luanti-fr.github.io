Téléchargements

Vous êtes un nouvel utilisateur ? Consultez le guide Pour bien débuter dans notre documentation. Sinon, n’hésitez pas à regarder le journal des modifications.

Windows

Fonctionne sous Windows 8, 8.1, 10 et 11. Peut fonctionner sur des versions plus anciennes, mais elles ne sont pas officiellement prises en charge.

  • Luanti 5.12.0 – version portable, 64 bits (recommandée)
  • Luanti 5.12.0 – version portable, 32 bits

Luanti est distribué sous forme d’archive portable .zip sur Windows. Extrayez-la dans un dossier où vous avez les droits d’écriture, puis lancez luanti.exe qui se trouve dans le dossier bin.

  • Paquet Luanti MSYS2 (maintenu par la communauté)

Vous pouvez aussi obtenir la dernière version en développement de Luanti depuis les builds réalisés par des membres de la communauté. Ces builds sont plus récentes que les versions officielles publiées et contiennent de nouvelles fonctionnalités (au prix d’une stabilité moindre).

Android

Android 6 ou version ultérieure est recommandé.

Vous pouvez également télécharger l’APK :

  • ARM (pour la plupart des appareils) : 64 bits – 32 bits
  • x86 : 64 bits – 32 bits

Note : Nous déconseillons d’utiliser les versions non officielles souvent trouvées sur le Play Store. Elles peuvent contenir des publicités excessives, des logiciels espions ou être distribuées sous des conditions propriétaires.

Linux

Paquets

Ces paquets peuvent parfois être obsolètes. Si c’est trop le cas, envisagez de compiler le logiciel à partir des sources.

  • Flatpak (toutes distributions) – Stable
  • Ubuntu et dérivés –
    • ajoutez le PPA (car les dépôts officiels sont obsolètes) pour la version stable : 
      sudo add-apt-repository ppa:minetestdevs/stable
    • ou pour la version quotidienne : 
      sudo add-apt-repository ppa:minetestdevs/daily-builds
    • puis installez Luanti : 
      sudo apt update
sudo apt install minetest
  • Debian – utilisez Flatpak ou compilez à partir des sources, car les dépôts officiels sont obsolètes
  • Arch Linux – Stable, Instable (AUR)
  • Fedora – Stable
  • openSUSE – Stable
  • Mageia – Stable
  • Gentoo – Stable
  • Void Linux – Stable
  • Alpine Linux – Stable
  • NixOS – Stable
  • Slackware – Stable (SBo)

FreeBSD

Gestionnaire de paquets

Sur FreeBSD 9.1 et versions ultérieures, vous pouvez utiliser les paquets précompilés officiels : pkg install luanti

Compilation avec les ports

Vous pouvez aussi compiler Luanti et choisir facilement les options de compilation via les dialogues des ports FreeBSD.

cd /usr/ports/games/minetest
make install

macOS

macOS 11.3 ou version ultérieure est recommandé. Peut fonctionner sur des versions plus anciennes s’il est compilé à partir des sources.

  • Apple Silicon signé – Luanti 5.12.0 – Application
  • Intel 64 bits signé – Luanti 5.12.0 – Application
  • Utilisez Homebrew pour obtenir la version stable :
    brew install --cask luanti
  • MacPorts contient aussi un port de la version obsolète :
    sudo port install minetest
