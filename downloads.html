Téléchargements

Vous êtes un nouvel utilisateur ? Consultez le guide Pour bien débuter dans notre documentation. Sinon, n’hésitez pas à consulter le journal des modifications.

Windows

Fonctionne sous Windows 8, 8.1 et 10. Peut fonctionner sur des versions plus anciennes, mais elles ne sont pas officiellement prises en charge.

  • Luanti 5.11.0 – version portable, 64 bits (recommandée)
  • Luanti 5.11.0 – version portable, 32 bits

Un problème ? Consultez l’aide pour installer Luanti sur Windows.

  • Paquet Luanti pour MSYS2 (maintenu par la communauté)

Vous pouvez également obtenir la version de développement la plus récente via les builds communautaires. Ces versions incluent de nouvelles fonctionnalités (mais peuvent être moins stables).

Android

Android 6 ou une version ultérieure est recommandé.

Vous pouvez également télécharger l'APK :

  • ARM (pour la plupart des appareils) : 64 bits - 32 bits
  • x86: 64 bits - 32 bits

Note : Nous déconseillons d'utiliser les versions non officielles disponibles sur le Play Store. Elles peuvent contenir des publicités excessives ou des logiciels espions, ou être distribuées sous des termes propriétaires.

Linux

Paquets

Ces paquets peuvent parfois être obsolètes. Si c'est le cas, envisagez de compiler le logiciel à partir des sources.

  • Flatpak (toutes distributions) – Stable
  • Ubuntu - PPA Stable ou PPA Daily, car les dépôts officiels sont obsolètes
  • Debian - utilisez Flatpak ou compilez à partir des sources, car les dépôts officiels sont obsolètes
  • Arch Linux - Stable
  • Fedora - Stable
  • openSUSE - Stable
  • Mageia - Stable
  • Gentoo - Stable
  • Void Linux - Stable
  • Alpine Linux - Stable
  • NixOS - Stable
  • Slackware - Stable (SBo)

FreeBSD

Gestionnaire de paquets

Sur FreeBSD 9.1 et versions ultérieures, vous pouvez utiliser les paquets précompilés officiels : pkg install minetest

Compilation avec les ports

Vous pouvez aussi compiler Luanti et choisir facilement les options de compilation via les dialogues des ports FreeBSD.

    cd /usr/ports/games/minetest
    make install

macOS

macOS 11.3 ou une version plus récente est recommandé. Peut fonctionner sur des versions plus anciennes si compilé à partir des sources.

  • Apple Silicon signé – Luanti 5.11.0 - Application
  • Intel 64 bits signé – Luanti 5.11.0 – Application
  • Utilisez Homebrew pour obtenir la version stable :
    brew install minetest
  • MacPorts contient également un port de la version stable :
    sudo port install minetest
