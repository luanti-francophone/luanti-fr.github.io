Servers

La liste des serveurs est aussi disponible sur servers.luanti.org.

Flags

Flag Explication
Pwd Les joueurs doivent définir un mot de passe lors de la première connexion.
Dmg Dommages activés
PvP Les joueurs peuvent endommager les autres joueurs
Cre Mode créatif

Annonce des serveurs

Définissez ce paramètre dans la configuration de votre serveur : 

server_announce = true
server_name = Mon serveur
server_description = Un serveur que j'ai fabriqué.
server_url = <URL de la page d'accueil du serveur ou d'un sujet de forum>
